## Estrutura do Projeto

 /nivel_basico/main.py  
  Implementa a estrutura fundamental de um RPG em Python, criando personagens, armas e monstros, mostrando status e simulações simples de ataques.

/nivel_intermediario/main.py  
  Adiciona inventário, poções e interação maior entre personagens (como uso de itens e dano automático ao atacar/alvo).

/nivel_avancado/main.py 
Sistema completo com habilidades especiais, fator de sorte que é (dado), sistema de batalha em turnos (um-contra-um e equipes), 
gerenciamento de inventário, uso de habilidades, golpes críticos, e batalhas entre grupos.

## O que foi feito

Todas as etapas do PDF foram feitas.
Criação e evolução das classes de personagens, armas, poções e monstros.
Adicionamos o inventário para armazenar itens.
Implementamos  métodos de ataque, cura e uso de habilidades.
criamos monstros especiais como Orc com golpe crítico.
Sistema de batalha por turnos, incluindo batalhas entre equipes com anúncio automático do vencedor.
